This slide has embedded R code, to demponstrate a dot product.


```r
# Define vectors x and y.
x <- c(1, -2, 3, 4, 5)
y <- c(-9, -8, 7, 6, -5)
```


```r
# Compute the product componentwise.
x * y
```

```
[1]  -9  16  21  24 -25
```

```r
# Compute the dot product.
(x %*% y)[1, 1]
```

```
[1] 27
```
